Data Access Instructions

1. If this dataset is locally available, it should be accessable at /programs/datagrid/#{identifier}

2. To download this dataset, please run the following command from your Terminal on a Linux or OS X workstation:

'rsync -av rsync://dv.sbgrid.org/#{fn:replace(globalId,"doi:","")} .' (Harvard Medical School, USA)

Depending on your location, faster access may be available from a Tier 1 site closer to your location

'rsync -av rsync://sbgrid.icm.uu.se/#{fn:replace(globalId,"doi:","")} .' (Uppsala University, Sweden)

'rsync -av rsync://sbgrid.pasteur.edu.uy/#{fn:replace(globalId,"doi:","")} .' (Institut Pasteur de Montevideo, Uruguay)

'rsync -av rsync://sbgrid.ncpss.org/#{fn:replace(globalId,"doi:","")} .' (Shanghai Institutes for Biological Sciences, China)

3. After the transfer is completed, please issue the following command to verify data integrity:

'cd #{identifier} ; shasum -c files.sha'
